How to customize JEWELRY easily #back to top

SASS General Options - #back to top

JEWELRY is developed in SASS. by using variable it's super easy to add changes. for compiling SASS we use Gurnt

CSS General Options - #back to top

We have add all major styles in style.css, and it's properly commented and very easy to customize. you can pick you block at top and easily search it in style.css file

Color Options - #back to top

JEWELRY HTML5 template has awesome features. in this template you have unlimited color option. it's super easy and fast to change color of complete site. we have created a separate css file called color.css which has all css. just open that file and replace your color code with existing one that's all, in result you will have you required color scheme on site

Font Options - #back to top

JEWELRY HTML5 template has awesome features. in this template you have unlimited Font option. it's super easy and fast to change Font of complete site. we have created a separate css file called fonts.css which has all css. just open that file and replace your Font family with existing one that's all, in result you will have you required Font family on site

Responsive Options - #back to top

JEWELRY HTML5 template has awesome features. in this template we have created a separate css file which has all Responsive style. it's super easy and fast to change any responsive in responsive.cc file.
